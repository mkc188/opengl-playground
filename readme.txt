CSCI3260 Assignment 2 Keyboard and Mouse Events

Demo procedure:
	Key "1": start seesaw motion;
	Key "2": swing the left swing;
	Key "3": swing the right swing;
	Key "4": clockwise rotatation of merry-go-round;
	Key "5": clockwise rotatation of merry-go-round;
    Key "6": change size of seesaw;
    Key "7": change the speed of animation;
    Key "8": toggle additional light;
	Key "0": cease object movement;
    Key "a": toggle global rotation;
    Key "r": reset to initial view;
    Key "d": toggle sky animation;
    Arrow Keys: human control;
    Mouse: drag to change view angle;
    Key "Esc": Exit
